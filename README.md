# fructosedb

## Building

Possible environment values:

- `USDA_KEY` - string for accessing USDA REST API
- `BACKEND_URL` - default: `http://localhost`
- `PORT` - default: `3000`

### Local Environment

Only `USDA_KEY` is required, can be supplied via `/.env` file.
